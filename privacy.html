1. Overview

This Privacy Policy explains how Vigil ("the Bot", "we", "us") collects, stores, uses, and protects data when used within Discord servers.

By adding or using Vigil, you agree to this Privacy Policy and the Terms of Service.

2. Data We Collect

  • Discord Server IDs
  • Channel IDs and Role IDs
  • User IDs
  • Audit log references for moderation tracking
  • Security event logs (e.g., rapid deletions, mass joins)
  • Configuration settings (whitelists, thresholds, permissions)
  • Server backup structure data (roles, channels, emojis, stickers, server name)

Vigil does not collect message content, direct messages, email addresses, passwords, or payment information.

3. How Data Is Used

  • Detecting raids and destructive activity
  • Performing automated moderation actions
  • Maintaining server backups
  • Providing security scoring and activity logs
  • Managing permission hierarchy

4. Automated Decision-Making

Vigil uses automated systems to detect suspicious behavior and may automatically quarantine, ban, or restrict users.

False positives may occur. Server administrators are responsible for reviewing and correcting automated actions.

5. Data Storage

Data is stored locally on the server hosting Vigil. Data files are stored in JSON format and are not encrypted by default.

It is the responsibility of the bot operator to secure the hosting environment and stored files.

6. Data Retention

Server configuration and logs remain stored unless manually deleted by the bot operator. Data is not automatically removed when a server removes Vigil.

7. Data Sharing

Vigil does not sell, trade, or share collected data with third parties except when legally required.

8. International Data Protection

Users and bot operators are responsible for complying with applicable data protection laws, including:

  • GDPR (European Union & UK)
  • CCPA (California, USA)
  • Other local data protection regulations

9. Security Measures

  • Securing server access with strong authentication
  • Limiting system access to trusted operators
  • Keeping hosting software up to date

10. Your Rights

Depending on your jurisdiction, you may have rights to request data access, correction, or deletion. Requests must be directed to the bot operator.

11. Changes to This Policy

We may update this Privacy Policy at any time. Continued use of Vigil after updates constitutes acceptance of the revised policy.

12. Contact

For privacy concerns or data-related questions, contact the bot operator through the platform where Vigil is deployed.
